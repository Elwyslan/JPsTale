# Priston Tale in Java
 Redevelop a 3D MMORPG "Priston Tale" in Java language, with free 3D game engine **jMonkeyEngine**


This project is aim to decode all the assets file in "Priston Tale". They are:

1. Fields
2. Items
3. Monsters
4. NPC
5. 3D models
6. Textures
7. Audios
8. Effects
9. Sky
